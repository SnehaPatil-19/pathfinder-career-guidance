#  PathFinder – AI Career Guidance System

##  Overview

PathFinder is an AI-powered career guidance system designed to help school students explore suitable career paths based on their interests, skills, and preferences. The system leverages cloud computing and intelligent logic to provide personalized recommendations in a simple and user-friendly interface.

---

##  Objectives

* Help students make informed career decisions
* Provide personalized career recommendations
* Demonstrate the use of cloud computing in real-world applications

---

##  Features

* 🔍 AI-based career suggestions
* 👩‍🎓 Student-friendly interface
* ☁️ Cloud-integrated backend
* ⚡ Fast and responsive system
* 📊 Personalized guidance based on inputs
